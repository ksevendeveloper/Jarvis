Requisitos do Sistema

Parâmetros e especificações mínimas necessárias para garantir a estabilidade e performance do pipeline do Jarvis.

Hardware Recomendado

  • CPU: 2 Cores VCPU (Mínimo) / 4 Cores (Recomendado para LLM local)
  • RAM: 1GB (Core backend) / 8GB (Para execução fluida do Ollama)
  • Armazenamento: 10GB de espaço livre em disco (SSD Opcional mas recomendado)

Sistema Operacional & Dependências

  • Debian 12 / Ubuntu 22.04 LTS (ou MacOS)
  • Python 3.10 ou superior
  • Node.js 18 (Para dashboards web/desktop options)

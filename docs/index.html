Jarvis
FastAPI + Socket.IO JWT + Policy + Allowlist Desktop FX Systemd Ready

Assistente Self-Hosted com Controle Seguro e Feedback Visual em Tempo Real

Jarvis combina backend robusto, autenticação JWT, validação por policy e experiências visuais no web dashboard e na máquina local.

Ler README Guia de Instalação Checklist Técnico

Highlights

  • Endpoint /api/execute com JWT obrigatório.
  • Bloqueio de padrões perigosos via core/conscience.py.
  • Allowlist de comandos via JARVIS_ALLOWED_COMMANDS.
  • Eventos Socket.IO para status de execução em tempo real.
  • Efeito visual local com ./cli.sh fx-on.

Quick Start

python3 -m venv .venv source .venv/bin/activate pip install -r requirements.txt PYTHONPATH=. python3 scripts/bootstrap_db.py --create-admin admin admin ./cli.sh start ./cli.sh fx-on

Diagramas Demonstrativos

Fluxo de Segurança

1. Usuário autentica e recebe JWT.
2. Socket.IO valida token no connect.
3. /api/execute valida JWT + policy + allowlist.
4. Backend executa e emite executing/success/error.
5. Dashboard e orb desktop reagem em tempo real.
